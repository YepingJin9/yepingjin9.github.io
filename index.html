Research

I'm broadly interested in optimization for learning, distributional robustness, and algorithmic foundations. Selected projects below.

Acceleration in Momentum Methods via Continuous-Time ODE Limits
Yeping (Patrick) Jin
Research project, 2022–present.
We derive ODE limits for Nesterov-style momentum, prove uniform convergence from discrete momentum to ODE solutions, and explain acceleration phenomena; experiments in PyTorch show 1–50× speedups over (stochastic) gradient descent on representative tasks.
arXiv / code
Randomized Algorithms for the Closest-Pair / Light-Bulb Problems
Yeping (Patrick) Jin and collaborators
Project lead, 2022.
Survey and analysis of subquadratic algorithms (e.g., LSH-based and ~O(n1.6) variants), connections to fast matrix multiplication, and deterministic variants for special cases; wrote a comparative review and empirical notes.
pdf / code
Second-Price Pacing Equilibria for Online Advertising
Yeping (Patrick) Jin and collaborators
Course project, 2022.
Constructed a pacing-augmented auction model using real CTR datasets; trained a decision-tree predictor and computed equilibria via MIP, perturbation, and smoothing methods (Gurobi); analyzed scaling behavior on semi-real data.
report / code
Expert-Guided Distributionally Robust Learning for Credible Health Predictions
Yeping (Patrick) Jin et al.
Ongoing work.
Formulates Wasserstein-robust learning with expert-structured regularization (group sparsity vs. expert-dense groups) for interpretable EHR prediction; includes synthetic and clinical evaluations.
